🐍 Retro-Klassiker Snake (Javascript)

🐍 Retro Classic Snake (Javascript)

HTML5 Canvas CSS3 Vanilla JS (ES6) LocalStorage

Ein responsives Browserspiel, das auf einem HTML5-Canvas gerendert wird. Es verfügt über eine präzise Steuerung (Sperrung entgegengesetzter Richtungen, um Selbstkollisionen durch schnelle Tastenfolgen zu vermeiden), Pausen- und Neustartfunktionen sowie eine persistente Highscore-Speicherung mittels LocalStorage.

A responsive browser game rendered on an HTML5 canvas. Features precise movement controls (locking opposite directions to prevent self-collision on rapid keyboard presses), start/pause/restart functions, and persistent high score storage using LocalStorage.

💡 Interaktives IT-Wissens-Quiz

💡 Interactive IT Knowledge Quiz

DOM Manipulation Dynamic HTML UX Design Accessibility

Ein barrierefreies Web-Quiz mit einer dynamischen Benutzeroberfläche. Fragen und Antworten werden dynamisch geladen. Das System bietet sofortiges visuelles UX-Feedback (Grün/Rot-Farben), sperrt Mehrfachantworten, zeigt den Lernfortschritt über eine animierte Leiste an und berechnet abschließend eine detaillierte Auswertung mit Feedback.

An accessible web quiz with a dynamic user interface. Questions and answers are dynamically injected. The system provides instant visual UX feedback (green/red colors), locks choices after submission, displays progress via an animated bar, and shows a detailed scorecard with dynamic feedback at the end.

💻 Modernes Web-Portfolio & Dashboard

💻 Modern Web Portfolio & Dashboard

CSS Grid / Flexbox Dark Mode API Client-side Filters Zweisprachiges System

Diese Web-Applikation dient als mein persönliches Portfolio. Sie vereint moderne Ästhetik (Glaseffekte, sanfte Schatten) mit technischer Raffinesse: ein dynamischer Suchfilter koordiniert mit Blog-Kategorie-Filtern, ein duales Theme-System (Dark/Light) und ein CSS-basiertes Togglesystem für nahtlose Mehrsprachigkeit (DE/EN) ohne redundante HTML-Seiten.

This web application serves as my personal portfolio. It combines modern design aesthetics (glassmorphism, soft shadows) with technical polish: a dynamic real-time keyword search coordinated with blog filters, a system dark/light mode toggle, and a CSS-based toggle system for seamless translations (DE/EN) without duplicate pages.
