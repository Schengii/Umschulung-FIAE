🍳 EcoChef — Nachhaltiges Koch- & Rezeptportal (Abschlussprojekt)

🍳 EcoChef — Sustainable Cooking & Recipe Portal (Capstone Project)

Java / Spring Boot REST API PostgreSQL Thymeleaf / CSS3 IHK Abschlussprojekt

EcoChef ist mein IHK-Abschlussprojekt. Die Webanwendung unterstützt Nutzer*innen dabei, gesunde und umweltfreundliche Mahlzeiten zu planen. EcoChef berechnet basierend auf den Zutaten automatisch einen ökologischen Fußabdruck (CO₂-Emissionen, Wasserverbrauch), schlägt regionale/saisonale Alternativen vor und bietet praktische Tipps zur Vermeidung von Lebensmittelverschwendung sowie zur energiesparenden Zubereitung.

EcoChef is my IHK capstone project. The web application helps users plan healthy and environmentally friendly meals. Based on the ingredients, EcoChef automatically calculates an ecological footprint (CO₂ emissions, water consumption), suggests regional/seasonal alternatives, and provides practical tips to avoid food waste and energy-saving cooking methods.
