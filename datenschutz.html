3. Local Storage

3. Local Storage

Diese Website verwendet den Local Storage Ihres Browsers, um folgende Einstellungen lokal auf Ihrem Gerät zu speichern:

This website uses your browser's Local Storage to save the following settings locally on your device:

  • Theme-Präferenz (Hell/Dunkel)
  • Theme preference (Light/Dark)
  • Sprachpräferenz (Deutsch/Englisch)
  • Language preference (German/English)
  • Benutzername (optional eingegebener Name für die Begrüßung)
  • Username (optionally entered name for the greeting)
  • Spiel-Highscores (Snake, Memory)
  • Game high scores (Snake, Memory)
  • Cookie-Zustimmung
  • Cookie consent

Diese Daten werden nicht an einen Server übertragen. Sie können den Local Storage jederzeit in Ihren Browser-Einstellungen löschen.

This data is not transmitted to any server. You can clear the Local Storage at any time in your browser settings.
